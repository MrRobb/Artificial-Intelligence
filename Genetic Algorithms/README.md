The algorithm tries to **evolve some individuals** in order to **match the phrase** that you introduce at the beginning of the program.


> You can specify:
>
>- Number of individuals
>
>- Mutation Rate
>
>- The phrase to guess
>

#### Usage

```sh
# Build
make build
# Run
make run
# Clean
make clean
# Help
make help
```
